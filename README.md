## 怎么用Course Hero才不亏？

买了会员只是第一步，关键是怎么用才能物超所值。

### 1. 别浪费你的30次解锁

每个月30次解锁听起来很多，但如果你不加思考地随便点开文档，很快就会用完。

我的建议是：**先浏览标题和预览，确定这份资料确实有用，再解锁**。

Course Hero的搜索功能很强大，你可以通过学校、课程、甚至具体的教授来筛选资料。花点时间找到最相关的那几份，而不是什么都点开看。

而且，解锁的资料只在30天内有效，不会累积到下个月。所以如果这个月你用不完，也不用心疼，该用就用。

### 2. 上传你自己的笔记，换免费解锁

Course Hero有个很聪明的机制：你上传10份文档，就能获得5次免费解锁。

这些文档可以是你自己的课堂笔记、学习指南、或者过往的考试复习材料。只要是原创的、对其他人有帮助的内容，都可以上传。

我有个同学就是靠这个方法，几乎没花钱就用了大半年。每次上完课，他都会把笔记整理一下上传，慢慢积累免费解锁次数。

不过要注意，**不要上传教授的原版PPT或者考试题**，这可能涉及版权问题，甚至违反学校的学术诚信政策。

### 3. 把导师问题留给最难的题

每个月40个导师问题听起来够用，但如果你每道题都去问，肯定不够。

我的策略是：**先自己尝试解决，实在搞不懂的再问导师**。

导师的回答质量通常很高，而且是一对一的详细解答，非常适合解决那种"卡住了完全没思路"的难题。但如果只是想确认一下答案对不对，其实看看已有的资料就够了。

### 4. 活用AI聊天功能

Course Hero的AI功能经常被忽视，但它其实超级好用。

你可以直接上传课程的PDF教材或者讲义，然后让AI帮你：
- 总结每个章节的重点
- 解释某个复杂的概念
- 生成复习问题

这个功能特别适合考前突击。我有次期中考试前一天才开始复习（不要学我），就是靠AI快速过了一遍三个章节的内容，最后还考了B+。

## 使用Course Hero的注意事项

这里必须严肃地说一下：**Course Hero是学习辅助工具，不是作弊工具**。

很多大学的学术诚信政策明确规定，直接抄袭网上的答案属于违纪行为。所以，正确的使用方式是：

1. **先自己尝试解题**，实在不会再去查资料
2. **把答案当作参考**，理解解题思路，而不是直接复制粘贴
3. **不要上传教授的原版材料**，比如考试题、作业答案等
4. **跟教授确认**学校是否允许使用这类平台

Course Hero的官方立场也很明确：他们不鼓励学生用平台作弊，平台的设计初衷是帮助学生更好地理解课程内容。

如果你用Course Hero只是为了应付作业、不是为了真正学习，那还不如不用。毕竟，大学学费那么贵，最后什么都没学到才是最亏的。
